About Chaq

The aim of this project is to create the most effective use of natural language software agents in implementing intelligent responses for human questions on a specific topic.

You can ask questoins about the following topics: .Net, Ajax, Android, Asp.Net, C, C#, C++, Css, Django, Git, HTML, Iphone, Java, Javascript, Jquery, JSON, Linux, MySql, Perl, PHP, Python, Ruby-on-Rails, SQL, Wordpress

Example questions:

  • What are common programming mistakes for PHP developers to avoid?
  • How to edit an incorrect commit message in GIT?
  • Why not use tables for layout in HTML?
  • How to undo last commit?
  • What is a metaclass in Python?
  • What is the best way to stop SQL injections in PHP?
  • Is Git better than Subversion?
  • How to check a checkbox with Jquery or Javascript?
  • What are the best resources to learn Javascript?
  • What does Ruby have that Python doesn't, and vice versa?
  • How can I remove a property from a Javascript object?
  • How do I check if a file exists using Python?
  • How to encode a URL in Javascript?

Software used in this project:

  • Python
  • Django
  • Natural Language Processing Python Toolkit
  • PyAIML
  • Stack Overflow (for the data)

You can see the proposal document for this project at the following URL: https://docs.google.com/document/pub?id=1CffvuiEYe3y7uXDdNxF6ioQY1Yp_w2qB__rfKiWAPxc
